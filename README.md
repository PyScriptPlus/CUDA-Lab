# CUDA Lab

A personal repository for learning, experimenting, and building projects with NVIDIA CUDA.

## Goals

- Learn CUDA programming
- Explore GPU parallel computing
- Build small CUDA examples
- Experiment with CUDA features
- Develop larger CUDA projects over time

## Requirements

- NVIDIA GPU
- NVIDIA Driver
- CUDA Toolkit
- C++ Compiler (GCC/Clang)

## Build

Compile a CUDA source file with:

```bash
nvcc main.cu -o main
```

Run:

```bash
./main
```
